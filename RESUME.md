## 项目描述

设计并实现一个基于 Multi-Agent + Agentic RAG 的电商智能客服系统。

系统支持：

- 商品价格查询
- 商品库存查询
- 订单物流查询
- 售后政策问答
- 多轮上下文追问

通过 SafetyAgent、IntentAgent、PlannerAgent、RetrievalRouter、ReflectionAgent 构建多 Agent 协作流程，实现任务规划、工具调用、证据反思和自动重规划。

- 设计并实现 Multi-Agent 电商智能客服系统。

- 实现商品库、订单库、售后政策 RAG 多工具动态路由。
- 引入 ReflectionAgent，实现证据校验与自动补任务。
- 基于 Memory 实现多轮上下文追问。
